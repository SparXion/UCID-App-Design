# Ubiquitous Language

| Term | Definition |
|------|------------|
| Student | A person enrolled at UC/DAAP. Owns Profile, Interests, Talents. |
| Talent | Innate ability (explicit/implicit). Dormant until developed. |
| Interest | Topic the student enjoys. |
| Skill | Learnable competency (e.g., "3D Sculpting"). |
| SkillTree | Graph of Skills forming a career cluster. |
| Subfield | Specialization in Industry (e.g., "Footwear Design"). |
| Industry | Market sector (e.g., "Consumer Products"). |
| Course | Learning resource tied to Skill. |
| CareerPath | Recommended sequence: Industry → Subfield → SkillTree → Skills. |
| HybridMode | One of: DIRECT_CREATOR, AI_CURATOR, SYSTEM_ARCHITECT, DESIGN_EXECUTOR |
| HybridType | One of: GENERATIVE, AUTOMATION, PROMPT_ENGINEERING, CONFIGURABLE_SYSTEMS |
| SystemBlurb | Marketing copy explaining "design the system that designs the product" |
